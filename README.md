# BRP Messenger

Децентрализованный P2P-мессенджер на базе протокола **BRP (Bauman Ring Protocol) v1.0**.

Протокол реализует обмен зашифрованными сообщениями и файлами в локальной сети с логической кольцевой топологией.
Каждый узел равноправен, данные защищены сквозным шифрованием (E2E) и цифровой подписью.
